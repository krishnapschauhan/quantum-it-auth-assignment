## Tech Stack
- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JSON Web Token (JWT)

---

## Features
- User Registration with MongoDB
- User Login with JWT authentication
- Secure password hashing using bcrypt
- JWT token and user data stored in browser localStorage
- Protected Dashboard route (cannot be accessed without login)
- Dashboard displays all registered users from the database
- Static UI fields (Role, Status, Action) implemented using dummy data as per assignment instructions
- Logout functionality
